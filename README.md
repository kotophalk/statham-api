# Statham API

Сборник «пацанских» мемов-цитат Джейсона Стэтхэма.
Проект включает парсер (на Python) для сбора цитат из открытых источников и готовый датасет в формате JSON.

## Использование
Запустите скрипт `statham_scraper.py` для сбора свежих данных.
Результаты сохраняются в `statham_quotes.json`.
